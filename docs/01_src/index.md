# JEMSSWrapper.jl

**JEMSSWrapper.jl** es un wrapper de JEMSS.jl, diseñado y pensado para facilitar específicamente la integración de algoritmos de neuroevolución para la optimización de políticas de relocalización dinámica de ambulancias.

## Características clave

* Mayor flexibilidad a la hora de implementar nuevas estrategias de relocalización. 
* Facilidad a la hora de crear replicas de escenarios y simulaciones con cambios en las estrategias de relocalización.
* Flexibilidad para experimentar con diferentes conjuntos de llamadas y ambulancias para un mismo escenario.
